How to share the screen?

Currently, this app do not support to share the screen. But, you can share the screen by 3rd party app!

1. OBS Studio (Windows)

Your Windows version should to be 64bit!

Now, let's Set up the OBS Vitual Cam

Let's read this requirement

- Supported Platforms : Windows 7 , Windows 8 and Windows 10
- Supported OBS Studio version : 24.0.0+ (Update to the Latest version of OBS, if you don't install OBS, please download this installation pack and install it: 64bit Installer or 32bit Installer )

Open OBS Virtual Cam Installation Pack (Download here)

Click Next to begin install

Click Next to accept the LICENSE

Choose save location and click Next to continue

Choose how much video output do you need, and click install!

You should to choose one (1)

Click Finish to close the Installation and restart the PC

Now, let's configure OBS Vitual Cam

Open OBS Studio, go to Tools and choose the VirtualCam
